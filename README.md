# Full Stack Apps on AWS Project

A NodeJS server with a REST API endpoint that processes incoming image URLs, deployed in AWS Elastic Beanstalk.

## Run the project

```
npm install
npm run start

```

## EB deployment

```
eb init
eb create

```
